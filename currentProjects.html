This is my current project at the moment. It follows a personified succulent bowl as it traverses the land in order to cultivate as many plants as it can and destroy the evil viruses and bacteria. I'm almost done with the game but I had to prune many of the features that I wanted to add due to bad framework design. jlib

I am currently working on a library coded in C that will make some parts of C++ easier and provide functionality/classes that the c++ stl does not provide. I'm building it in C because I would like it to be something that I can use if I choose to use C for a project too and because it makes exporting it into a library easier.

This was inspired by the lack of graphs and graph algorithms in the C++ stl and because I'm currently going through an algorithm text book and thought it would be good to implement some of them.

You can find this library at github.com/nanoblob-jpeg/jlib
