Tetris Remake

I made this because I wanted an interactive way to practice code and I thought that recreating a classic game within a time limit would be a fun way to do it. This was my introduction into using JavaFX and making interactive applications. However, it's implementation is far from what would be used for a game nowadays. My implementation uses a timing function in order to simulate frames and changes the color of squares that are placed in a grid. This results in a very slow update and input capture loop.

If you would like to try out this remake of tetris for yourself, you can download it by clicking here. Feel free to email me for the source code too though I will warn you that it is not the cleanest program out there.
